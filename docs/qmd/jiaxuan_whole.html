3 Fine-Gray Models — eGFR Ratio (CR-CYS / CG)

egfr_ratio = ckd_epi_gfr_cre_cys_unindex / cockcroft, entered as a linear continuous predictor in a Fine-Gray competing-risk model (death is the competing event). Adjuster set: age + sex + bmi + smoking + steroids + thyroid + tumor_stage + ecog_score_cat2 + Paclitaxel + Pemetrexed + Gemcitabine + ici + score. Anemia models add pre_HGB_45days; thrombocytopenia models add pre_PLT_45days; AKI models add pre_CRE_45days; composite models add both baseline HGB and PLT.

Code 
fg_model <- function(time_var, status_var, pred_var, pre_labs = NULL) {
  adj <- c("age","sex","bmi","smoking","steroids","thyroid","tumor_stage",
           "ecog_score_cat2","Paclitaxel","Pemetrexed","Gemcitabine",
           "ici","score")
  if (!is.null(pre_labs)) adj <- c(adj, pre_labs)
  fmla <- as.formula(paste0(
    "Surv(", time_var, ", ", status_var, ") ~ ",
    paste(c(pred_var, adj), collapse = " + ")
  ))
  tidycmprsk::crr(fmla, failcode = 1, data = dat) %>%
    tbl_regression(exp = TRUE,
                   pvalue_fun = function(x) style_pvalue(x, digits = 3)) %>%
    add_n()
}

6 Model Diagnostics

6.1 Events per Variable (EPV)

Models with EPV < 10 are at risk of instability; EPV < 5 should not be reported without penalisation.

Code 
outcomes <- c("anemia_2", "anemia_3", "throm_2", "throm_3",
              "aki_2", "aki_3", "grade_2_aes", "grade_3_aes")

epv_table <- purrr::map_dfr(outcomes, function(out) {

  status_col <- paste0("cmp_status_", out)
  n_pred <- case_when(
    grepl("anemia", out)  ~ 15L,
    grepl("throm", out)   ~ 15L,
    grepl("aki", out)     ~ 15L,
    TRUE                  ~ 16L
  )
  tibble(
    outcome      = out,
    N            = sum(!is.na(dat[[status_col]])),
    events       = sum(dat[[status_col]] == 1, na.rm = TRUE),
    deaths       = sum(dat[[status_col]] == 2, na.rm = TRUE),
    n_predictors = n_pred,
    EPV          = round(events / n_pred, 1)
  )
})
